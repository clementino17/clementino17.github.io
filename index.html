Data Visualization

Visualisasi data adalah suatu proses untuk mempresentasikan data ke dalam bentuk visual agar mudah dipahami. Di kursus ini kita akan mempelajari tentang penggunaan tools visualisasi seperti Tableau, PowerBI, R Shiny dan Plotly Python

Coding

Coding merupakan proses atau kegiatan pengolahan kode yang kamu tuliskan menggunakan bahasa pemrograman tertentu. Ada tiga macam bahasa yang wajib dikuasai, yaitu: SQL, Shell Scripting, Python dan R.

Statistic

Statistika adalah ilmu yang digunakan untuk mengumpulkan, mengelola, menganalisis, dan mempresentasikan data, sehingga bisa diambil kesimpulan. Kita juga menggunakan ilmu statistik ini guna mempelajari Machine Learning yang akan diterangkan oleh tutor kami.

Business

Domain knowledge tiap lini bisnis sangat diperlukan untuk menyelesaikan persoalan data science. Sementara itu Kemampuan umum dalam bisnis mencakup tiga hal berikut: 1. Kemampuan presentasi (Presentation Skill) 2. Kemampuan manajemen proyek 3. Kemampuan berinteraksi dan komunikasi (People skill)
